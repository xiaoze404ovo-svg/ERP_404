# ERP_404 - 人力资源管理系统

基于企业 ERP 数据库开发的人力资源管理系统，专为企业人事部门设计。

## 功能模块

- **员工信息档案管理** - 员工档案的增删改查，支持22个字段完整信息
- **部门与岗位架构管理** - 组织架构树形展示，部门增删改
- **考勤记录追踪** - 打卡记录查询与月度考勤汇总
- **薪酬福利管理** - 工资数据查询，支持多科目转置显示
- **人事变动记录** - 入职/离职/调岗/晋升等变动跟踪
- **数据可视化仪表盘** - Chart.js 图表展示统计数据
- **报表导出** - CSV 格式数据导出

## 技术栈

- 后端: Python Flask
- 前端: Bootstrap 5 + Chart.js + jQuery
- 数据库: SQLite (基于现有 ERP 数据库)

## 快速启动

```bash
cd hrms
pip install flask flask-cors openpyxl
py app.py
```

访问 http://127.0.0.1:5000
